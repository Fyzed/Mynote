第一步：安装cnpm（https://npm.taobao.org/）

	npm install -g cnpm --registry=https://registry.npm.taobao.org


第二步：

	cnpm install --global gulp

第三步：

	新建一个项目文件夹

	切换到d盘：D:
	切换到项目目录: cd mobile_app
	gulp安装到项目目录：cnpm install --save-dev gulp

第四步：

	运行gulp ：No gulpfile found


第五步：

	cnpm install browser-sync --save-dev

第六步：

	新建gulpfile.js保存到项目根目录文件夹

第七步：
	
	运行 gulp browser-sync
